Open Logbook v1.0

The Open Logbook specification aims to be an open interchange format for flying applications on handheld devices to sync easily with one another. This specification aims to be flexible and powered by JSON.

The specification provides a preamble, a section where the format version, a UNIX timestamp when the export was generated, a title and by what software generated the file.

The flights section is a list of flights to be imported or exported by the application. A flight will contain:

  • UNIX timestamp when the flight was flown
  • A "to" field that marks the destination of the flight
  • A "from" field that marks the origin of the flight
  • An optional "enroute" field that describes the flight's enroute details as a list
  • An Aircraft Section that describes the aircraft used for the flight
  • A "remarks" field that holds a list of remarks for the flight
  • A Fields Section the holds key-value pairs for the flight details

Aircraft

Describes an aircraft used for a flight. An aircraft must contain the following fields, any additional metadata can be used.

  • A "type" field used to identify the aircraft type
  • A "tailNumber" field that is the aircraft's tail number
  • A "remarks" field that can be used to hold remarks about an aircraft

Fields

The keys and values in the "fields" section describe the details for each individual flight. Valid field keys are:

  • PIC type: decimal - total Pilot In Command time
  • SIC type: decimal - total Second In Command time
  • totalTime type: decimal - Total Time
  • approaches type: integer - number of Approaches
  • takeoffs type: integer - number of Takeoffs
  • landings type: integer - number of Landings
  • picCrossCountry type: decimal - total PIC Cross Country time
  • complex type: decimal - total Complex airframe time
  • nightTakeoffs type: integer - number of Night Takeoffs
  • nightLandings type: integer - number of Night Landings
  • multiengine type: decimal - total Multi-engine airframe time
  • crossCountry type: decimal - total Cross Country time
  • night type: decimal - total Night flying time
  • instrument type: decimal - total Instrument flying time
  • simulatedInstrument type: decimal - total Simulated Instrument flying time
  • dualReceived type: decimal - total Dual Received time
  • dualGiven type: decimal - total Dual Given time
  • hobbs type: decimal - Hobbs ending time
  • tach type: decimal - Tach ending time
